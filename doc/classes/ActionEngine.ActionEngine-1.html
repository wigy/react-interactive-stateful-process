Static handle

  • handle<ActionType>(action: ActionType | ActionType[], props: RenderingProps<Setup, Element>): ActionResult
    • Defined in ActionEngine.ts:57

    Processor for a triggered action on the given element.

    Type parameters

    • ActionType: DebugAction = DebugAction

    Parameters

    • action: ActionType | ActionType[]
    • props: RenderingProps<Setup, Element>

    Returns ActionResult

    The element in the props is checked for action definitions. If there is no actions defined, the result is success. If there is a single action, it is executed and the resulting value is returned. If there is an array of actions, all of them are executed. If any of them fails, the result is failure. Otherwise success.
